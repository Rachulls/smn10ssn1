FUNDAMENTOS JS

PROTOTIPOS

Un prototipo es un mecanismo para compartir propiedades y metodos entre objetos.

Prototipo String: Es cuando declaramos un valor en palabras/letras.

Prototipo Number: Es cuando declaramos un valor numerico.

Prototipo Boolean: Es cuando declaramos un valor true o false.

ARRAYS

Son objetos similares a una lista.

Index(indexOf): Este metodo retorna el indice en el que se encuentra un elemento de un array.

Push(): El metodo push agrega elementos al final del array original.

Pop(): El metodo pop elimina el ultimo elemento de un array.

OBJETOS

Un objeto es una estructura de datos que almacena y organiza informacion.

Un objeto literal es cuando se crea y se define un objeto sin usar clases. Se denota usando {}.

Assign: Con el método Object.assign() se agregan propiedades a un objeto.

Mostrar info: Se usa el mètodo Object.values(), esto devuelve el array con los valores en orden.

Destructuring: Permite dfescomponer objetos en partes pequeñas.

This: Hace referencia al objeto que contiene el método donde se invoca.

FUNCIONES

Es un bloque de código reutilizable que realiza una tarea específica.

Declaracion: Es la forma más tradicional de definir una función, utilizando la palabra clave 'function'.

Expresion: Se refiere a la definición de una función como parte de una expresión y asignarla a una variable o a una propiedad de un objeto.

Arrow function: Proporciona una sintaxis más concisa para definir funciones
